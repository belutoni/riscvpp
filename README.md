# riscvpp
RISC-V Assembler and Emulator written in modern C++23.

## Key Features
* <b>No</b> dependencies.
* Cross-Platform

## Testing
Using CTest.

## AI Usage
* AI Generated Code is prohibited from appearing in the Assembler or Emulator!
* Tests are currently generated with Gemini Flash/Pro with Extended Thinking.
